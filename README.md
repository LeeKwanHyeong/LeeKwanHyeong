## 2. forecaster_lib
시계열 수요예측 모델의 학습, 추론, 데이터 로딩, 체크포인트 관리를 하나의 재사용 가능한 **Python Forecasting Library**로 정리한 개인 개발 프로젝트입니다.

### Key Points
- `modeling_module` 패키지 기반 시계열 forecasting library 구조 설계 및 구현
- `train`, `predict`, `load_predictor`, `build_dataloader` 중심의 public API 구성
- `TrainRequest`, `DataRequest`, `TrainerConfig`, `ArtifactConfig` 등 dataclass 기반 request interface 설계
- PatchTST, PatchMixer, Titan, ExoTST 계열 모델 학습 및 추론 registry 구성
- point / quantile / distribution forecasting 학습 경로 정리
- endogenous-only 및 past/future exogenous one-table data loading 구조 구현
- warm-up stage / spike stage 기반 staged training 구조 설계
- intermittent demand weighting, spike-aware sampling, horizon decay 등 수요예측 특화 학습 옵션 실험
- checkpoint 기반 inference runtime 및 training manifest artifact 구조 정리
- public / private module boundary를 분리하여 패키징 및 private wheel 배포 구조 설계
- 테스트 코드 기반 public API, checkpoint, model registry, device resolution 검증 흐름 구성

### Tech Stack
`Python` `PyTorch` `Pandas` `Polars` `Scikit-learn` `Time Series Forecasting` `Deep Learning` `Packaging` `PyTest`
